# Первая программа
**простой код на C++**

```c++

#include <iostream>
#include <cstdlib> // для system
using namespace std;

int main() { 
    cout << "Hello, world!" << endl;
    system("pause"); // Только для тех, у кого MS Visual Studio
    return 0; 
```
Каждая строчка в коде на языке программирования C++ оканчивается символом ";".. это очень важно!!!

Директива #include используется для подключения других файлов в код.
без using namespace std; ( using namespace std - указывает на то, что мы используем по умолчанию пространство имен с названием «std» )

В коде всегда идет функция по умолчанию main(), int - тип данных возращаемого значения ( в питоне это было по типу if __name__ == "__main__": )

Оператор **cout** предназначен для вывода текста на экран командной строки.
После него ставятся две угловые кавычки (<<). Далее идет текст, который должен выводиться.
Он помещается в двойные кавычки. Оператор **endl** переводит строку на уровень ниже. Но endl не явл-ся полной копией "\n" потому как endl еще очищает буфер.

```c++

#include <iostream>

int main() { 
    std::cout << "Hello, world!" << std::endl;
    return 0; 
}
```
т.к пространство имен std не подключен => исп-ся конструкция std:: для обращения к методам std. Например:


```C++
std::cout << "hi";
std::endl;
```

По мимо cout std включает в себя множество других методов ( которые есть в офиц.документации ). Например std::cin() - предназначен для ввода значения в определенную переменную.. потом идет символ ">>" и затем название переменной в которую необходимо вставить значение. 

```C++
int x;
std::cin >> x;
```
более красиво так:
```C++
int x;
std::count << "Введите переменную x: ";
std::cin >> x;
```

# Переменные и типы данных + объявление переменной.
Из школьного курса математики мы все знаем, что такое переменные. В программировании принципы довольно схожи. Переменная — это «ячейка» оперативной памяти компьютера, в которой может храниться какая-либо информация.

В программировании переменная, как и в математике, может иметь название, состоящее из одной латинской буквы, но также может состоять из нескольких символов, целого слова или нескольких слов.

В языке С++ все переменные имеют определенный тип данных. Например, переменная, имеющая целочисленный тип не может содержать ничего кроме целых чисел, а переменная с плавающей точкой — только дробные числа.

Тип данных присваивается переменной при ее объявлении или инициализации. Ниже приведены основные типы данных языка C++, которые нам понадобятся.
Основные типы данных в C++

    1. int — целочисленный тип данных.
    2. float — тип данных с плавающей запятой.
    3. double — тип данных с плавающей запятой двойной точности.
    4. char — символьный тип данных.
    5. bool — логический тип данных.
    
Объявление переменной в C++ происходит таким образом: сначала указывается тип данных для этой переменной а затем название этой переменной. Пример объявления переменных:

```C++
int a; // объявление переменной a целого типа. 
float b; // объявление переменной b типа данных с плавающей запятой. 
double c = 14.2; // инициализация переменной типа double. 
char d = 's'; // инициализация переменной типа char. 
bool k = true; // инициализация логической переменной k.
```

1. Заметьте, что в C++ оператор присваивания (=) — не является знаком равенства и не может использоваться для сравнения значений. Оператор равенства записывается как «двойное равно» — ==.

2. Присваивание используется для сохранения определенного значение в переменной. Например, запись вида a = 10 задает переменной a значение числа 10.

Переменная также может быть константной ( иметь фиксированное значение ):
```C++
const int x = 10;
const double pi = 3.14;
```

Также константу можно объявить через директиву #define. #Define создает макрос, который представляет собой ассоциацию идентификатора или параметризованного идентификатора со строкой токена. После определения макроса компилятор может подставить строку токена для каждого обнаруженного идентификатора в исходном файле.
В программах могут встречаться повторяющиеся фрагменты кода. Для того, чтобы вызвать часто используемые фрагменты кода, в программах на C++ можно использовать:

        1. функции;
        2. макросы
Для объявления макроса используется директива #define. Общая форма объявления макроса следующая:
```C++
#define MacroName(parameters) expression
```
, где 
    MacroName – имя макроса;
    parameters – параметры, которые получает макрос;
    expression – выражение, которое вычисляет макрос. Выражение размещается той же строке, что и директива #define.

Пример:
```C++
#define PI 3.14
#define X 10
```

# Операции над переменными
Основные арифметические операции ( отн-ся к бинарным операциям ):
  1. "+" – сложение;
  2. "-" – вычитание;
  3. "*" – умножение;
  4. "/" – деление;
  5. "%" – остаток от деления.

Арифметические операции можно применять для:

    1. целочисленных типов: short int, unsigned short int, int, unsigned int, long, unsigned long;
    2. типов с плавающей запятой (вещественных типов): float, double, long double;
    3. типов (классов), которые содержат «перегруженные» арифметические операции.

В арифметических операциях один из операндов может иметь целочисленный тип, а второй быть типом с плавающей запятой. В этом случае результат имеет тип с плавающей запятой.

Унарные операции (+, -) означающие знак переменной.

Пример кода:
```C++
int a, b;

a = -8; // унарная операция '-', обозначает знак числа
b = +9; // унарная операция '+', b = 9
a = b-5; // бинарная операция '-', используется для вычисления выражения
```

# Логические операции
В языке программирования C/C++ используются следующие логические операции:

    1. && – логическое «И»;
    2. || – логическое «ИЛИ»;
    3. ! – логическое «НЕТ».
![изображение](https://user-images.githubusercontent.com/86359412/201086497-84b8045e-f9bb-4c6d-b89e-db5ece2332fb.png)

Пример исп-я:
```C++
// логические операции
bool res;
int a, b;

// операция && (AND)
a = 8;
b = 5;
res = a && b; // res = True

a = 0;
res = a && b; // res = False

// операция || (OR)
a = 0;
b = 0;
res = a || b; // res = False

b = 7;
res = a || b; // res = True

// операция ! (логическое "НЕТ")
a = 0;
res = !a; // res = True

a = 15;
res = !a; // res = False
```

Комментарии в языке C++ обозначаются так: 1) однострочные "// ... ", 2) многострочные """ ... """ 

Язык С/С++ поддерживает следующие поразрядные логические операции:

    1. & – поразрядное логическое И (AND);
    2. ^ – поразрядное сложение по модулю 2 (XOR — исключающее ИЛИ);
    3. | – поразрядное логическое ИЛИ (OR);
    4. ~ – поразрядная инверсия (NOT).
    
Каждый бит результата определяется на основе двух операндов, которые являются битами, так как показано в таблице.

![изображение](https://user-images.githubusercontent.com/86359412/201086914-524f61a6-22a6-424a-902b-d910ef6ef5e7.png)

Язык С/С++ включают две операции поразрядного сдвига:

    1. << – сдвиг влево значения операнда на заданное количество бит. Операнд размещается слева от знака операции. Число сдвигаемых бит указывается справа от знака операции;
    2. >> – сдвиг вправо значения операнда на заданное количество бит. Операнд размещается слева от знака операции (<<). Количество сдвигаемых бит размещается справа от знака операции.
    
Пример поразрядного сдвига:
```C++
// Операции сдвига
int a;
int b;
int c;

a = 15;
b = -5;

// сдвиг влево - умножение
c = a << 1; // c = a * 2^1 = 30
c = b << 2; // c = b * 2^2 = -20

// сдвиг вправо - деление
c = a >> 3; // c = a / 2^3 = 1
c = b >> 1; // c = b / 2^1 = -3
```
Также можно реализовать лог-ю операцию XOR. Как известно, язык C++ не содержит встроенной логической операции XOR (исключающее «ИЛИ»).

Ниже приведен фрагмент кода, который реализует операцию XOR с помощью использования операций && (AND), || (OR), ! (NOT).

```C++
// реализация операции XOR через операции && (AND), || (OR), ! (NOT)
bool x, y;
bool res;

...

res = (x || y) && !(x && y);

...

В вышеприведенном программном коде:

    1. x, y – переменные, для которых вычисляется операция XOR;
    2. res – переменная, которая есть результатом вычисления.

```

Также очень важно понимание инкримента ( ++ ) и дикремента ( -- )
```C++
int i = 0;
++i; // i = 1
i++; // i = 0 знач изменяется не сразу
```

sizeof() - это унарный оператор, возвращающий длину в байтах переменной или типа, помещенных в скобки. 

# Операторы Break и Continue
Оператор break предназначен для искусственного прерывания выполнения:

    последовательности итераций в операторах цикла for, while или do-while;
    последовательности операторов в операторе выбора switch.

Чаще всего оператор break используется в сочетании с оператором условия if. 

Оператор continue предназначен для перехода к выполнению следующей итерации цикла. Если в теле цикла встречается оператор continue, то:

    выполнение текущей итерации останавливается;
    происходит переход к следующей итерации цикла.

Оператор continue может быть применен во всех видах циклов: for, while, do-while.

В большинстве случаев вызов оператора continue осуществляется при выполнении некоторого условия в операторе if.
